# Report and a Personal Perspective About the Challenge

## About Mistakes

It was my first try on the challenge, and I think I committed some mistakes that I can fix in another try, but they put me in a bad position in my first attempts:

- I prioritized cleaning the dataset, and I didn't study how the variables were interacting with each other and didn't visualize an EDA. Maybe it was my biggest mistake, because the data quality could have been increased with some decisions that would have been made before.
- My code could be more divided, because at one point, there was one notebook to do all cleaning from the raw df to the cleaned df ready for the model. And it was hard to organize when the process became bigger.

## Improvement

At the moment, I prefer not to watch tutorials end-to-end, because it would make the challenge meaningless. So, to avoid this, I will read and study carefully the tutorials that Kaggle provides in the overview of the challenge.
